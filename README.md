# Chart-racing

EChart + Nuxt(Vue2)で Youtube にありがちな棒グラフを実現する

## デバッグ起動

```bash
yarn install
yarn dev -o
```

## 使い方

- tsv ファイルを用意する
- start ボタンで開始

## TSV ファイル仕様

- 具切り文字：`\t`
- 改行文字：`\n`
- ファイル末尾の改行：なし

### 列ヘッダ（１行目）

| 列名      | 説明                                                                               |
| --------- | ---------------------------------------------------------------------------------- |
| sliceName | チャート右下に表示される時系列タイトルの列であることを示す。大文字、小文字問わない |
| {other}   | データ系列名                                                                       |

### 色定義（２行目）

文字列で色定義する`#FFFFFF`や`rgba(r, g, b, a)`などの形式

### 3 行目移行

棒グラフのデータ
